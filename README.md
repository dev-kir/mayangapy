A simple python web scrapper to retrieve prayer times in Kuala Terengganu zone from www.e-solat.gov.my, www.waktusolat.my, www.waktusolat.xyz.

The output format:
{
  "asar": "16:37",
  "imsak": "06:01",
  "isyak": "20:31",
  "maghrib": "19:21",
  "subuh": "06:11",
  "zohor": "13:22"
}

call it by:
/prayer-times-esolat
/prayer-times-waktusolat
/prayer-times-waktusolatxyz
